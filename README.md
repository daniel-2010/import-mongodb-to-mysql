# Import MongoDB to MySQL
Este código é um exemplo de como importar os registros contidos dentro de um Banco de Dados MongoDB para um Banco MySQL. 
Para realizar a importação dos registros foi utilizado a tecnologia <b>NodeJS</b> como intermediário entre os dois bancos.

Execute os comandos a seguir:

Para instalar os pacotes do projeto:

<b>$ npm install</b>

Comando para executar o código:

<b>$ npm run dev </b>

A pasta <u>/assets<u> possui os bancos em <b>.json</b> e <b>.sql</b> para a realização dos testes.
